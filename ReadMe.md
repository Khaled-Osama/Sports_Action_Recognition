# Sports Action Recognition

this is a model for recognizing the human action recognition (sports). by the help of Deep learning we have built a high accuracy model that can differntiate between 10 classes so, we by passing the input to the model we can know what's the sports in that video and what is the human do.


## Required Installations

the are some of installations needed to run the system successfully, we need to install the following:

```
$ pip install tensorflow
$ pip install dm-sonnet
$ pip install opencv-contrib-python
$ pip install numpy
$ pip install tqdm
$ pip install pandas
$ pip install matplotlib
$ pip install seaborn
```

## Sample code

Run the example code using
```
python evaluate_sample
```

the script outputs the string describe what action in the scene.
